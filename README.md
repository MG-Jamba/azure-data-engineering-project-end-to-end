Azure Data Engineering Project

## My Overview
This Project demonstrates an end to end data pipeline using Azure services.

## Technologies Used
- Azure Data Factory implementing a robust data pipeline and design
- Azure Data Lake
- Databricks for data intergration and transformation
- Azure Synapse Analytics for efficient data warehousing and analytics
- Apache Spark for best practices handling big data solutions and real time processing

- ARMTemplate parameters for factory.json
- Factory/
- LinkedTemplates/
